About:

RQ Mods is a toolbox full of useful mods for the RQ repair ticket management software.

Here's what it can do:

Features

  • "Custom quick link:" adds a customizable link to the top navigation bar in RQ.
  • "Serial number reminders:" when editing a ticket, highlight the serial number if it is shorter than five characters. Reminds employees to go back and fill in the full serial number if they previously entered something like "N/A."
  • "Quote reminders:" when viewing a customer page, highlight the "Quotes" link on the left if the customer has a quote available to convert.
  • "Inventory-view click target adjustment:" when looking at inventory search results, clicking anywhere except for the checkbox or menu button doesn't do anything.
  • "Ticket-view click target adjustment:" when looking at a list of tickets, clicking anywhere except for the ticket number doesn't do anything.
  • "Pattern recorder" and "Pattern printer:" adds a graphical canvas for drawing customer's unlock patterns. This can be found in the right NAVIGATION sidebar. The Printer feature draws the unlock pattern on the label-printing page, so you can print out the pattern along with the usual barcode label.

RQ fixes

  • "Catalog search drop down fix:" in the current version of RQ, the dropdown menu that appears when searching for a catalog item while editing a ticket floats off the screen to the right. This fix re-positions that box so it always fits on-screen.
  • "Quick search focus:" when you open the quick search menu, automatically focus the cursor inside the search box.
